# Flipbook Catalog

Turn your PDFs into a browsable catalog library on your WordPress site — no coding required.

Flipbook Catalog adds a dedicated content type for your PDF publications. Upload a PDF, give it a title and featured image, and your visitors get a beautiful interactive flipbook powered by the [Flipbook Block](https://github.com/philhoyt/flipbook-block) plugin. Everything is wired up automatically — no shortcodes, no page builders, no fiddling with templates.

## What It Does

- **Dedicated content type** — keeps your catalogs, magazines, or brochures organised separately from posts and pages.
- **Media library integration** — select any PDF straight from your WordPress media library with a familiar upload interface.
- **Automatic flipbook template** — single catalog pages use a pre-built block theme template that displays the flipbook automatically. No template editing required.
- **Configurable labels and URLs** — call it "Magazines", "Lookbooks", "Brochures" — whatever fits your brand. Change the label and URL slug from a simple Settings page.

## Requirements

- WordPress 6.9 or higher
- PHP 8.1 or higher
- [Flipbook Block](https://github.com/philhoyt/flipbook-block) plugin (free, installed separately)
- A block-based theme

## Installation

1. Install and activate the [Flipbook Block](https://github.com/philhoyt/flipbook-block) plugin.
2. Download this plugin and upload the `flipbook-catalog` folder to `wp-content/plugins/`, or install it directly from the WordPress admin.
3. Activate **Flipbook Catalog** from the Plugins screen.
4. A new **Catalogs** menu item will appear in your WordPress admin. Start adding your PDFs.

## Getting Started

1. Go to **Catalogs → Add New**.
2. Give your catalog a title and featured image.
3. In the **PDF File** box, click **Upload / Select PDF** and choose your file from the media library.
4. Publish. Your catalog is live with a fully interactive flipbook.

To customise the content type name or URL, go to **Catalogs → Settings**.
